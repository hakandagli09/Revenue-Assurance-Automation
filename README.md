How It Works (High-Level)

1. Data Ingestion
Pulls raw ERP sales orders and multi-provider commission statements.

2. Data Standardization
Normalizes inconsistent fields, provider names, and confirmation codes using configurable mappings.

3. Reconciliation Engine
Matches orders to commissions, calculates gaps, and categorizes discrepancies.

4. Automated Outputs
Exports categorized tabs, provider KPIs, and a clean dataset for visualization.

5. BI Layer (Tableau)
Leadership accesses interactive dashboards built on reconciled, trustworthy data.
